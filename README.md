# jQuery FormStyler <sup>[4.0.2](https://github.com/alexanevsky/formstyler/blob/master/CHANGELOG.md)</sup>

jQuery plugin for HTML forms: inputs and selects (include multiple selects):

- `<input type="checkbox">`
- `<input type="radio">`
- `<input type="file">`
- `<input type="number">`
- `<select>`
- `<select multiple>`


## Quick start

Just add:
```html
$('input, select').formstyler();
```
